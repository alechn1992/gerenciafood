# gerenciafood

Projeto de gestão para food service (em construção).

## Sobre
Repositório sincronizado via GitHub para permitir trabalho a partir de
qualquer máquina com o Claude Code instalado.

## Como trabalhar de outra máquina
```bash
git clone https://github.com/alechn1992/gerenciafood.git
cd gerenciafood
# abra o Claude Code nesta pasta
```

Ao terminar as alterações:
```bash
git add -A
git commit -m "descrição da mudança"
git push
```
